# SATS

Projeto front-end modular do Super Auxiliar Tecnico de Seguranca.

## Uso local

Abra `index.html` diretamente no navegador. Para melhor compatibilidade com bibliotecas externas, tambem e possivel servir a pasta com qualquer servidor HTTP local simples.

## Estrutura

- `index.html`: entrada principal, login, central e Plano de Acao.
- `procedimentos.html`: visualizacao interna isolada carregada pelo index.
- `css/`: estilos separados por responsabilidade.
- `js/`: logica separada por dominio, usando scripts tradicionais em ordem controlada.
- `assets/`: imagens e demais arquivos estaticos.
